栈和队列

2018-09-17

基础

第一篇博客就水一下栈和队列吧~~望轻喷=

栈：

栈操作网上各种解释（百度）千奇百怪~甚是佩服;
简而言之，栈就是一个有缺陷的数组，你只能在数组的最后一个元素进行删除和添加;
窝的栈一般就是a[++t]=value(添加元素)/t--（删除元素）;
栈简单实用,居家旅行必备（当然得有用才行2333
栈除了一些裸题外，一般还可以应用于深搜的非递归版本(~当然我会选择写递归);

简单介绍栈的一个特性，也就是栈的出栈入栈顺序的总数符合卡特兰数(Catalan);
何谓出栈入栈顺序，简而言之：比如当n=3是有三个数，姑且叫1 2 3，辣么可以有如下的可能：
1 2 3 3 2 1
1 2 2 3 3 1
1 2 2 1 3 3
1 1 2 2 3 3
1 1 2 3 3 2
 (每个数字第一次出现代表入栈，第二次出现代表出栈);
一种五种，即第三个卡特兰数为5
1 1 2 5 14 42 132 429....（卡特兰数第0项开始~~~;
至于求解方式有递推公式:
a[n]=a[0]*a[n-1]+a[1]*a[n-2]+...+a[n-1]*a[0];(a[0]=1,n>=1);

然(其)后(实)我(是)就(只)不(会)再(这)废(么)话(点);

队列：

借鉴上面栈的说法，队列也可以看成一个有缺陷的数组，都只能在数组末尾添加元素，不同的是，队列只能在数组开头删除;
可以用俩元素l,r分别代表元素的开头和结尾下标，添加元素即a[++r]=value,删除的话就是l++;
当然这是栈最朴素(笨)的实现方式，极易发现这样很费空间，即每次删除操作就会产生一个多于的空间不再使用，因此就有了循环队列:

循环队列基本构造和队列一样，就是加个l,r极值的限制（姑且限制为max)，当进行添加操作时使得r>max，那么就使得r=1;
相信很多人有疑问了，现在状态下的区间[l,r]的元素不对了————没错，现在队列里的元素实际上是[1,l]+[r,max];
同时易证循环队列里有且只有这两种状态，辣么好办，直接拿一个变量标记下就ok（可能有人考虑直接把lr比大小就直接能分离两种状态,但是l==r时却是不好判断;

上面说了栈可以用于深搜，那么它的好基友队列自然就应用于广搜了;
